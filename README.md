# Google-photos-backup
Manually backup Google Photos sources on a schedule.

# Usage: 
./google-photos-backup.sh [install|uninstall|schedule|manual-sync]
  install      - Install and configure gphotos-sync
  
  uninstall    - Remove gphotos-sync and cleanup
  
  schedule     - Configure or update backup schedule
  
  manual-sync  - Run a manual sync now
